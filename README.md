Electric Vehicle Price Prediction

Machine Learning project to predict EV prices using Python and Scikit-learn.

Project Overview

This project analyzes electric vehicle data and builds ML models to predict EV prices based on specifications like battery capacity, range, top speed, and brand.

Features

- Exploratory Data Analysis (EDA) with visualizations
- Multiple ML models: Linear Regression, Decision Tree, Random Forest, Gradient Boosting
- Model comparison and evaluation
- Price prediction for new EVs
